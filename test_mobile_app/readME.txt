Файлы разделены по страницам и модулям, js файлы лежат в папке src/js и название файлов соответствуют страницам, собираются файлы через webpack, исходные файлы(js) и готовые файлы(js) указаны в файле webpack.config.js. Страницы находятся в папке pages. Стили находястя в папке dist/css, так же они разделены на общие (для модулей) и стили которые нужны определенным страницам которые лежат в папке название которой соответствует странице.
